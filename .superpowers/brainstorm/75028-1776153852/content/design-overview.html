File Changes

Modified Files
lib/shaders/fragment.glsl — raymarching SDFs, parallax UV offset, 3D projection function (~300 lines)

lib/shaders/vertex.glsl — MVP transform, noise displacement (~40 lines)

lib/engine.ts — grid mesh creation, VAO swap, MVP matrix computation, new uniform setters

lib/store.ts — new 3D state fields (global + per-layer depth)

lib/layers.ts — add depth to LayerParams

components/GradientPanel.tsx — 3D shape controls section

components/EffectsPanel.tsx — parallax + mesh distortion sections

components/LayerPanel.tsx — depth slider per layer
New Files
lib/math.ts — mat4 helpers (perspective, lookAt, rotate) — ~80 lines, no dependencies

No other new files needed. Everything integrates into existing modules.
